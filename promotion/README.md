# Dossier Promotion - E.S.C BELITEI

Ce dossier contient tous les contenus marketing pour promouvoir le site et l'entreprise.

## Structure

```
promotion/
├── README.md                    # Ce fichier
├── reseaux-sociaux/
│   ├── post-facebook-instagram.md   # Posts pour Facebook et Instagram
│   └── post-linkedin.md             # Posts pour LinkedIn
├── google-my-business/
│   └── fiche-entreprise.md          # Fiche GMB + posts
└── publicite/
    ├── google-ads.md                # Campagnes Google Ads
    └── facebook-instagram-ads.md    # Campagnes Meta Ads
```

## Utilisation

### Réseaux sociaux
1. Copier le texte du post souhaité
2. Ajouter les photos du dossier `public/images/`
3. Adapter le numéro de téléphone et l'URL si nécessaire
4. Publier !

### Google My Business
1. Se connecter sur [business.google.com](https://business.google.com)
2. Compléter la fiche avec les informations fournies
3. Publier les posts régulièrement (1-2 par semaine)
4. Ajouter les photos des réalisations

### Publicité en ligne
1. **Google Ads** : Créer un compte sur [ads.google.com](https://ads.google.com)
2. **Meta Ads** : Créer un compte sur [business.facebook.com](https://business.facebook.com)
3. Suivre les guides dans chaque fichier
