# Guide pas-à-pas : Créer sa première campagne Google Ads

## Prérequis

- Un compte Google (Gmail)
- Une carte bancaire (pour le paiement)
- Le site web en ligne (www.belitei.fr)
- Budget : minimum 5-10€/jour pour commencer
- 45 minutes de disponibilité

---

## Étape 1 : Créer un compte Google Ads

1. Aller sur **https://ads.google.com**
2. Cliquer sur **"Commencer"**
3. Se connecter avec le compte Google (Gmail)

### Premier écran : Objectif

Google va demander l'objectif principal. Sélectionner :
**"Générer des prospects"** (appels, formulaires de contact)

### Nom de l'entreprise
```
E.S.C BELITEI
```

### Site web
```
https://www.belitei.fr
```

Cliquer sur **"Suivant"**

---

## Étape 2 : Passer en mode Expert (IMPORTANT)

Par défaut, Google propose le mode "Intelligent" (simplifié) qui est moins efficace.

1. En bas de l'écran, chercher **"Passer en mode Expert"**
2. Cliquer dessus
3. Confirmer

> **Pourquoi ?** Le mode Expert donne plus de contrôle sur les enchères, les mots-clés et le ciblage.

---

## Étape 3 : Choisir le type de campagne

Sélectionner : **"Créer une campagne sans objectif spécifique"**

Puis choisir le type : **"Recherche"**

> C'est le type le plus efficace pour un artisan : vos annonces apparaissent quand quelqu'un cherche "électricien Épinay" sur Google.

Cliquer sur **"Continuer"**

---

## Étape 4 : Paramètres de la campagne

### Nom de la campagne
```
Recherche - Électricité Île-de-France
```

### Réseaux

Cocher :
- ✅ Réseau de Recherche Google

Décocher (pour commencer) :
- ❌ Partenaires du Réseau de Recherche
- ❌ Réseau Display

> **Conseil** : Commencer uniquement avec la Recherche Google pour mieux contrôler le budget.

---

## Étape 5 : Ciblage géographique

### Zone de ciblage

1. Sélectionner **"Saisir un autre lieu"**
2. Cliquer sur **"Recherche avancée"**
3. Choisir **"Cibler un rayon"**
4. Entrer :
   - Lieu : **Épinay-sur-Seine, France**
   - Rayon : **30 km**

**OU** cibler par départements :

Ajouter un par un :
- Paris (75)
- Seine-Saint-Denis (93)
- Hauts-de-Seine (92)
- Val-d'Oise (95)
- Val-de-Marne (94)

### Options de ciblage géographique

Sélectionner : **"Présence : personnes situées dans vos zones ciblées ou qui s'y rendent régulièrement"**

> Évite de payer pour des clics de personnes qui ne sont pas dans votre zone.

---

## Étape 6 : Langues

Sélectionner : **Français**

---

## Étape 7 : Budget quotidien

### Pour commencer (test) :
```
10 €/jour
```

Cela représente environ **300€/mois**.

### Budget recommandé après test :
```
15-20 €/jour
```

> **Conseil** : Commencer petit, analyser les résultats pendant 2 semaines, puis augmenter si ça fonctionne.

---

## Étape 8 : Stratégie d'enchères

Pour débuter, sélectionner :
**"Maximiser les clics"**

Cocher : **"Définir une limite d'enchère au CPC maximale"**
Entrer : **2,00 €**

> Cela évite de payer trop cher pour un seul clic.

**Après quelques semaines** (quand vous aurez des conversions), passer à :
**"Maximiser les conversions"**

---

## Étape 9 : Créer un groupe d'annonces

### Nom du groupe d'annonces
```
Électricien Île-de-France
```

### Mots-clés

Copier-coller cette liste :

```
électricien île de france
électricien épinay sur seine
électricien 93
électricien seine saint denis
installation électrique
rénovation électrique
tableau électrique
mise aux normes électriques
dépannage électrique
électricien paris
```

### Types de correspondance (IMPORTANT)

Par défaut, Google met les mots-clés en "Requête large" (moins précis).

Pour plus de contrôle, modifier en **"Expression exacte"** :

```
"électricien île de france"
"électricien épinay sur seine"
"électricien 93"
"installation électrique"
"rénovation électrique"
"tableau électrique"
```

> Les guillemets indiquent une expression exacte.

Ou en **Mot-clé exact** (très précis) :

```
[électricien épinay sur seine]
[électricien 93]
```

> Les crochets indiquent un mot-clé exact.

## Étape 10 : Créer les annonces

### Annonce 1 : Électricité générale

**URL finale :**
```
https://www.belitei.fr
```

**Titres (max 30 caractères chacun) :**
```
Titre 1 : Électricien Île-de-France
Titre 2 : Devis Gratuit sous 24h
Titre 3 : E.S.C BELITEI
Titre 4 : Installation & Dépannage
Titre 5 : Artisan Qualifié
Titre 6 : Intervention Rapide
```

> Ajouter au moins 5-6 titres. Google les combine automatiquement.

**Descriptions (max 90 caractères chacune) :**
```
Description 1 : Électricien qualifié en Île-de-France. Tableau, prises, éclairage LED. Devis gratuit 24h.

Description 2 : Installation et rénovation électrique. Artisan de confiance à Épinay-sur-Seine. Appelez !

Description 3 : Mise aux normes NF C 15-100. Travail soigné et garanti. Intervention rapide.
```

**Chemin affiché :**
```
Chemin 1 : electricite
Chemin 2 : devis-gratuit
```

L'URL affichée sera : `belitei.fr/electricite/devis-gratuit`

## Étape 11 : Extensions d'annonces (TRÈS IMPORTANT)

Les extensions augmentent la visibilité et le taux de clic. **Ne pas sauter cette étape !**

### Extension de lieu

1. Cliquer sur **"Extensions"** > **"Extension de lieu"**
2. Lier le compte Google My Business
3. L'adresse apparaîtra dans l'annonce

### Extension d'appel

1. Cliquer sur **"Extension d'appel"**
2. Entrer le numéro :
```
[NUMÉRO DE GHEORGHII]
```
3. Les gens pourront appeler directement depuis l'annonce

### Extensions de liens annexes

Ajouter 4 liens :

**Lien 1 :**
- Texte : `Nos Services`
- URL : `https://www.belitei.fr/#services`
- Description : `Électricité, plomberie, VMC, carrelage`

**Lien 2 :**
- Texte : `Nos Réalisations`
- URL : `https://www.belitei.fr/#realisations`
- Description : `Découvrez nos chantiers terminés`

**Lien 3 :**
- Texte : `Demander un Devis`
- URL : `https://www.belitei.fr/#contact`
- Description : `Devis gratuit sous 24h`

**Lien 4 :**
- Texte : `Nous Contacter`
- URL : `https://www.belitei.fr/#contact`
- Description : `Appelez ou envoyez un message`

### Extensions d'accroche

Ajouter des accroches (petits textes) :
```
Devis Gratuit
Intervention Rapide
Artisan Qualifié
Travail Garanti
20 ans d'expérience
Paiement Facilité
```

### Extension d'extrait de site

- En-tête : **Services**
- Valeurs : `Électricité, Plomberie, VMC, Carrelage, Rénovation`

## Étape 12 : Vérifier et publier

1. Vérifier le récapitulatif de la campagne
2. Cliquer sur **"Publier la campagne"**

### Configurer le paiement

1. Entrer les informations de facturation
2. Ajouter une carte bancaire
3. Définir un seuil de paiement (ex: 50€)

> Google débitera quand le seuil est atteint ou en fin de mois.

---

## Étape 13 : Configurer le suivi des conversions

**TRÈS IMPORTANT** pour mesurer les résultats !

### Conversion 1 : Appels

1. Aller dans **"Outils et paramètres"** > **"Conversions"**
2. Cliquer sur **"+ Nouvelle conversion"**
3. Sélectionner **"Appels téléphoniques"**
4. Choisir **"Appels vers un numéro de téléphone sur votre site Web"**
5. Configurer le suivi

### Conversion 2 : Formulaire de contact

1. **"+ Nouvelle conversion"**
2. Sélectionner **"Site Web"**
3. Action : **"Envoyer un formulaire de prospect"**
4. Installer le tag sur la page de remerciement

> **Note** : Cette étape est technique. Si besoin, je peux t'aider à ajouter le code sur le site.

---

## Étape 14 : Ajouter des mots-clés négatifs

Les mots-clés négatifs empêchent vos annonces d'apparaître pour des recherches non pertinentes.

1. Aller dans **"Mots-clés"** > **"Mots-clés à exclure"**
2. Ajouter :

```
gratuit
pas cher
formation
emploi
recrutement
stage
salaire
métier
devenir
cours
DIY
faire soi-même
tuto
tutoriel
```

> Cela évite les clics de personnes qui cherchent des infos, pas un artisan.

## Optimisation après lancement

### Semaine 1-2 : Observer

1. Aller dans **"Campagnes"** chaque jour
2. Vérifier :
   - Nombre de clics
   - Coût par clic (CPC)
   - Nombre d'appels/conversions
   - Termes de recherche réels

### Rapport "Termes de recherche"

1. Aller dans **"Mots-clés"** > **"Termes de recherche"**
2. Voir les vraies recherches des utilisateurs
3. Ajouter en négatif les termes non pertinents
4. Ajouter en mots-clés les bons termes manquants

### Semaine 3-4 : Optimiser

1. Désactiver les mots-clés qui ne convertissent pas
2. Augmenter les enchères sur les mots-clés performants
3. Tester de nouvelles annonces
4. Ajuster le budget si nécessaire

## Indicateurs à surveiller

| Métrique | Bon résultat | Action si mauvais |
|----------|--------------|-------------------|
| CTR (taux de clic) | > 3% | Améliorer les titres |
| CPC (coût par clic) | < 2€ | Réduire les enchères |
| Taux de conversion | > 5% | Améliorer la page |
| Coût par conversion | < 30€ | Optimiser les mots-clés |

---

## Erreurs à éviter

1. ❌ **Budget trop faible** - Moins de 5€/jour ne donne pas assez de données
2. ❌ **Ciblage trop large** - Toute la France = budget gaspillé
3. ❌ **Pas de mots-clés négatifs** - Clics inutiles et coûteux
4. ❌ **Une seule annonce** - Toujours tester 2-3 variantes
5. ❌ **Ignorer les termes de recherche** - Vérifier chaque semaine
6. ❌ **Pas de suivi des conversions** - Impossible d'optimiser
